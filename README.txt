청렴 에너지 히어로 Mobile Demo v5.5

실행 방법
- index.html을 Chrome, Safari, Edge 등 최신 브라우저에서 엽니다.
- 웹 서버에 업로드하면 스마트폰에서도 링크로 바로 실행할 수 있습니다.

PC 조작
- 이동: A/D 또는 방향키
- 점프: Space
- 공격: Z
- 방패: X
- 정화 미사일: C

모바일 조작
- 왼쪽 하단: ◀ / ▶ 이동
- 오른쪽 하단: ⚔ 공격 / 🛡 방패
- 버튼을 누르고 있는 동안 이동이 유지됩니다.
- 가로 화면 플레이를 권장합니다.

v5.5 변경사항
- 모바일 터치 컨트롤 추가
- 멀티터치 지원(이동 중 공격/방패 가능)
- 화면 크기에 맞춘 반응형 캔버스
- 스크롤, 더블탭 확대, 길게 누르기 메뉴 방지
- 모바일 전용 조작 안내 문구
- PC 키보드 조작도 그대로 유지
